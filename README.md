# nendoban : 粘土板 : Bootstrap 4 ベースのブランクWordPressテーマ

Bootstrap 4ベースで特に機能的なものは入っていない使いやすいテーマがあったほうがいいかな？と思って作ったテーマ

こねくり回す土台のイメージで「粘土板」という名前にしました。
